About

저는 자기주도적 학습을 통해 꾸준히 성장하는 백엔드 개발자입니다.

Java와 Spring Boot를 기반으로 견고한 백엔드 시스템을 구축하고, Node.js와 NestJS로 확장 가능한 아키텍처를 설계합니다.

데이터베이스 설계부터 API 구현까지 전체 백엔드 개발 사이클을 경험하고 있으며, 효율적인 코드와 깨끗한 아키텍처를 지향합니다.
